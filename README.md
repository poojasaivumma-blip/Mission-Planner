# Mission-Planner
🚀 Mission Tracker — Space-Themed To-Do Application
📌 Project Overview

Mission Tracker is a space-themed task management web application built using HTML, CSS, and Vanilla JavaScript. The application helps users organize tasks as “missions” with priority levels, deadlines, completion tracking, and real-time progress monitoring.

The UI is designed with a futuristic space aesthetic to enhance user engagement and provide an intuitive task management experience.

🎯 Objectives
Build a functional and interactive to-do application using core web technologies.
Implement task tracking with status management (active/completed).
Introduce deadline-based overdue detection.
Provide visual feedback using progress indicators and UI animations.
Maintain a clean, responsive, and modern UI design.
🧩 Key Features
1. 🪐 Space-Themed UI Design
Gradient blue cosmic background
Futuristic Orbitron font for title
Clean glass-like mission cards
Hover micro-interactions for better user experience
2. 📌 Mission Management System

Each task is treated as a mission with the following attributes:

Mission title (user input)
Deadline (date picker)
Priority level (Low / Medium / High)
Completion status (checkbox)
Delete option
3. ⚡ Priority System
Priority is represented using a colored side indicator bar
No textual color labeling is used to maintain UI clarity:
Low → Green indicator
Medium → Yellow/Orange indicator
High → Red indicator
4. 📊 Progress Tracking
Real-time progress bar based on completed missions
Percentage completion displayed below the progress bar
Active mission counter shown above the progress bar
5. ⏰ Overdue Detection System
Missions automatically evaluated based on selected deadline
Tasks past their due date are marked as overdue
Overdue missions are visually highlighted with a red glow border for quick identification
6. 🎮 Interactive UI Enhancements
Hover animations on mission cards
Button scaling and glow effects
Input field focus animations
Smooth transitions for progress updates
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling, layout, animations, and theme design
JavaScript (Vanilla JS) – Application logic and DOM manipulation

⚙️ How It Works
User enters a mission name.
User selects:
Deadline (date)
Priority level
Mission is added to the list on clicking “Accept Mission”
User can:
Mark mission as completed
Delete mission
Application automatically:
Updates progress bar
Calculates active missions
Detects overdue tasks
